# Customer Churn & Revenue Leakage Analytics

## Overview
This project analyzes customer churn patterns to identify revenue leakage and actionable retention strategies using SQL, Python, and Tableau.

## Objectives
- Identify churn drivers
- Quantify revenue loss
- Segment high-risk customers
- Build an executive-ready dashboard

## Tech Stack
- PostgreSQL / MySQL
- SQL (CTEs, Window Functions, Views)
- Python (Pandas, SQLAlchemy)
- Tableau

## Status
Project initialized. Database design in progress.
